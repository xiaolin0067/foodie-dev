题目
详解布隆过滤器
位图(Bit Map)

n=样本量，p=失误率
m=布隆过滤器的bit位数，k=hash函数个数

m = - (n * ln p) / (ln 2)^2
k = (ln2 * (m / n)) / (0.7 * (m / n)) 向上取整
p(真) = (1 - e^(- (n * k(真)) / m(真))) ^ k(真)
